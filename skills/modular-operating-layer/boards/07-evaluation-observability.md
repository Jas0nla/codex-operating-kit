# Board Registry Entry

- `board_name`: `evaluation-observability`
- `owner_agent`: `评估与观测官`
- `required_context`: `board state`, `property source surfaces when applicable`, `handoff outcomes`, `agent return quality`, `rework traces`
- `forbidden_context`: `raw full-thread memory`, `creative draft content unless needed for defect diagnosis`, `unlabeled source mixing`
- `success_signal`: `visible quality signal on where the system is holding or degrading, plus a property fact packet when the task is listing content`
- `handoff_fields`: `board stage`, `owner explicit`, `fact packet status`, `last handoff status`, `latest rework cause`, `source surfaces used`

# Rules

- Team-local entry: `$Master`
- Strong multi-lane team-local task: `$team-orchestrator`
- Ask once before multi-agent launch
- Default shared roster: `workflow_scout` + `repo_researcher`
- Add `final_verifier` only for the last credibility pass
- Route before delegate
- One write scope, one owner
- Risky delegated writes need a plan first
- No nested teams
- Prefer known local helpers, scripts, env files, stable IDs, and entrypoints before generic exploration
